## Get started

<Steps>
<Step title="Get your API key">

Sign up at [plantstore.dev](https://plantstore.dev) and grab your API key from the dashboard.

</Step>

<Step title="Make your first request">

```bash
curl -X GET "https://api.plantstore.dev/v3/plant/1" \
  -H "api_key: YOUR_API_KEY"
```

</Step>

<Step title="Explore the docs">

Browse the [API Reference](/api-reference) for detailed endpoint documentation, request/response schemas, and code examples in multiple languages.

</Step>
</Steps>
