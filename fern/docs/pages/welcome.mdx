### Try the API

<CodeBlocks>
  ```python title="Python"
  from plantstore import PlantStore

  with PlantStore("<apiKey>") as client:
      plant = client.plants.get(plant_id=1)

      print(f"{plant.name}")
      print(f"  Status: {plant.status}")
      print(f"  Tags: {', '.join(plant.tags)}")
  ```
  ```bash title="cURL"
  curl -X GET "https://api.plantstore.dev/v3/plant/1" \
    -H "api_key: YOUR_API_KEY"
  ```
  ```javascript title="Node.js"
  import { PlantStoreClient } from "@plantstore/sdk";

  const client = new PlantStoreClient({ apiKey: "<apiKey>" });

  const plant = await client.plants.get({ plantId: 1 });

  console.log(plant.name);
  console.log(`  Status: ${plant.status}`);
  console.log(`  Tags: ${plant.tags.join(", ")}`);
  ```
</CodeBlocks>
